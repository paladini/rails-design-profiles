Rails design infrastructure

Design intent, made operational.

Keep AI design references close. Turn only the decisions you trust into versioned CSS tokens.

Install the gemHow it works
# config/design_profiles.yml
active: editorial
profiles:
  editorial:
    tokens:
      color-primary: "#c15f3c"
      font-body: "Newsreader, serif"

$ bin/rails design:activate[editorial]
Active design profile: editorial

DESIGN.md is a reference. CSS variables are a contract. Your Rails components remain yours.

01

Bring a reference

Install an openly licensed DESIGN.md reference or add your own. Give agents visual context without coupling it to production CSS.

02

Choose tokens

Translate the decisions that fit your product into a small, reviewable YAML profile.

03

Switch with confidence

Activate a profile. The gem emits standard --rdp-* variables; your components decide how to use them.

A smaller surface area for better design change.
